# Blob intensity measurements

**Type:** Property Worker  
**Category:** Blobs  
**Annotation Shape:** `polygon`  
**Interface Category:** Intensity  

## Description

This tool computes the pixel intensity of objects in the specified channel. It will compute the mean, max, min, median, 25th percentile, and 75th percentile intensity, as well as the total intensity. Learn more

## Interface Parameters

| Parameter | Type | Required | Default | Range / Options | Description |
|-----------|------|:--------:|---------|-----------------|-------------|
| **Channel** | `channel` | Yes | -- | -- | Compute pixel intensities in this channel. The channel does not have to be the same as the layer the annotations are on. |
| **Z planes** | `text` |  | -- | -- | Z positions to compute intensities for (empty to use annotation plane) |
| **Additional percentiles** | `text` |  | -- | -- | Percentiles to compute intensities for (leave empty for default of 25, 75) |
