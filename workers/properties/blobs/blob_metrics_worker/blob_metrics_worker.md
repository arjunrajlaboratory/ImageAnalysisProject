# Blob metrics

**Type:** Property Worker  
**Category:** Blobs  
**Annotation Shape:** `polygon`  
**Interface Category:** Morphology  

## Description

This tool computes a variety of metrics for the objects in the specified channel. The metrics computed are: area, perimeter, centroid, compactness, elongation, convexity, solidity, rectangularity, circularity, fractal dimension, and eccentricity.

## Interface Parameters

| Parameter | Type | Required | Default | Range / Options | Description |
|-----------|------|:--------:|---------|-----------------|-------------|
| **Use physical units** | `checkbox` |  | -- | -- | If checked, the metrics will be computed in physical units. If unchecked, the metrics will be computed in pixels. |
| **Units** | `select` |  | µm | m, mm, µm, nm | The units of the physical units. If unchecked, the metrics will be computed in pixels. |
