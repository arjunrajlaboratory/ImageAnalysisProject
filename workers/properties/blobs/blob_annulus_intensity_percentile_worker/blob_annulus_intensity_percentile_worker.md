# Annulus intensity percentile measurements

**Type:** Property Worker  
**Category:** Blobs  
**Annotation Shape:** `polygon`  
**Interface Category:** Intensity  

## Description

This tool computes the pixel intensity in an annulus around the objects in the specified channel at the specified percentile. For instance, if you set the percentile to 90, it will compute the 90th percentile intensity of the annular region. The size of the annulus is defined by the radius.

## Interface Parameters

| Parameter | Type | Required | Default | Range / Options | Description |
|-----------|------|:--------:|---------|-----------------|-------------|
| **Channel** | `channel` | Yes | -- | -- | Compute pixel intensities in this channel. The channel does not have to be the same as the layer the annotations are on. |
| **Radius** | `number` |  | 10 | 0 - 200 |  |
| **Percentile** | `number` |  | 50 | 0 - 99.99999 |  |
