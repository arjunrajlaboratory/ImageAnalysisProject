# Connection IDs

**Type:** Property Worker  
**Category:** Connections  
**Annotation Shape:** `any`  
**Interface Category:** Connections  

## Description

This tool adds a property to the objects to document the connections between them, which is helpful for time-lapse analysis. It gives each object an ID, and then shows the ID for each object's parent and child. If there is no parent or child, it assigns -1.
