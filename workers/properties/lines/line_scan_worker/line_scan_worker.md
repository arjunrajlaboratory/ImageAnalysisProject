# Line scan CSV

**Type:** Property Worker  
**Category:** Lines  
**Annotation Shape:** `line`

## Description

This tool computes the intensity along a line and puts the results in a CSV file. The intensity is computed for all channels by default, but this can be restricted to a single channel. The output is saved as a CSV file in the dataset folder.
