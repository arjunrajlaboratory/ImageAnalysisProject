# Laplacian of Gaussian

**Type:** Annotation Worker  
**Interface Category:** Object Detection  

## Description

This tool finds spots in an image using the Laplacian of Gaussian method.It uses a filter to enhance spots, then uses a threshold to find the spots.It can work in 2D (current z-slice) or 3D (z-stack).
