# Connect Time Lapse

**Type:** Annotation Worker  
**Interface Category:** Connections  

## Description

This tool connects objects across time slices. It allows you to connect objects even if there are gaps in time. Learn more

## Interface Parameters

| Parameter | Type | Required | Default | Range / Options | Description |
|-----------|------|:--------:|---------|-----------------|-------------|
| **Object to connect tag** | `tags` |  | -- | -- | Connect all objects that have this tag. |
| **Connect across gaps** | `number` |  | 0 | 0 - 10 Time pt | The size of the time gap that will be bridged when connecting objects across time. |
| **Max distance** | `number` |  | 20 | 0 - 1000 pixels | The maximum distance (in pixels) between the child and parent objects to be connected. Otherwise, objects will not be connected. |
